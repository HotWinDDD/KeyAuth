# KeyAuth
Minecraft plugins for protect your server

# How To Use
 ### 1. Download Plugin
 ### 2. Copy to the 「plugins」 folder
 ### 3. Start you server
 ### 4. Enjoy the game ❤

# Command
 ### /key [password] - Login The Server
 ### /keyreload - Reload Plugin/config
 ### /keyinfo - Check password update time
